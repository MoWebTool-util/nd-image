# nd-image

[![spm version](http://spmjs.io/badge/nd-image)](http://spmjs.io/package/nd-image)

> 图片缩放、居中、旋转、加载。

## 安装

```bash
$ spm install nd-image --save
```

## 使用

```js
var Image = require('nd-image');
// use Image
```
## 开发

### 本地 Web 服务

```bash
grunt
```

浏览器中访问 http://127.0.0.1:8851

### 生成/查看 API 文档

```bash
grunt doc
grunt
```

浏览器中访问 http://127.0.0.1:8851/doc

### 代码检查与单元测试

```bash
grunt test
```

### 发布组件到 SPM 源

```bash
grunt publish
```
